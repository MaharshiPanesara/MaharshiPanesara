1. what is software?
->Software refers to a collection of computer programs, data, and instructions that tell a computer how     to perform specific tasks or functions.

2.what is software Engineering?
->Software engineering is a systematic and disciplined approach to designing, developing, testing, and    maintaining software systems.

3. explain types of software?

-> types:

                1. System Software:-         
                -> Operating system (OS)        
              	-> ex: Android, ios, windows, linux, etc..
       
              	2. Application Software:-         
              	(i) Desktop Application         
              	-> ex: VS code, notepad, firefox, etc..
         
              	(ii) Mobile Application       
              	-> ex: Instagram, netflix, snap, etc..
        
              	(iii)Web Application        
              	-> ex: Web browser
         
              	3. Programming Software:-      
              	-> A program that can be written in any programming language to perform specific task.         	
              	-> Compiler         
              	-> Interpreter
         
              	4. Deriver Software:-       
              	-> ex: bluetooth deriver, audio deriver, etc..
       
              	5. Middleware Software:-       
              	-> It's an intermediary software between two or more applications/software.         
              	-> ex: database
